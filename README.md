#  Heart Disease Prediction App

A simple Machine Learning web app built with Streamlit to predict heart disease risk.

##  Features
- Predicts heart disease risk
- Easy-to-use interface
- Shows prediction probability

##  Inputs
Age, Gender, Chest Pain Type, Blood Pressure, Cholesterol, Fasting Blood Sugar, ECG, Max Heart Rate, Exercise Angina, Oldpeak, Slope, Vessels, Thal

##  Tech Stack
Python, Streamlit, Scikit-learn, Pandas, NumPy

##  Run

```bash
pip install -r requirements.txt
streamlit run app.py
